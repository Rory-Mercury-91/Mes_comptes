Sections et champs

Chaque section a un titre et un type d'affichage : Inline (une ligne par champ) ou Liste (puces <li>). Utilisez les flèches ▲ ▼ pour réordonner les champs et les sections. Le type Consigne importante génère un bandeau rouge. Le bouton + Image insère le code <img> de l'alarme orange. Cochez Réductible sur une section ajoutée pour un bloc <details>/<summary>. Couleurs et styles : panneau Mise en forme. Décochez le lien vidéo ou le texte du bas si vous n'en avez pas besoin. Le lien vidéo reste toujours juste au-dessus de la section connexion vidéo et bouge avec elle. Le lien fichier (PDF, etc.) s'affiche en fin de section Informations diverses : saisissez l'URL sur le serveur, ou utilisez Parcourir… pour reprendre le nom du fichier. Utilisez Exporter / Charger les informations pour sauvegarder ou reprendre vos saisies.

Texte de la remarque

Rédigez votre remarque avec l'éditeur ci-dessous. Le HTML généré contiendra uniquement ce texte, sans sections ni lien vidéo.

Mise en forme

Réglez pour chaque partie : couleur du texte, surlignement (fond), taille, gras, italique et souligné. Les titres de section apparaissent automatiquement ici.
