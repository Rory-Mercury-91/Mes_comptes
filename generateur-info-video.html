Générateur Remarques

Créez vos sections, remplissez les champs, réglez la mise en forme et copiez le HTML généré.

Sections et champs

Chaque section a un titre et un type d'affichage : Inline (une ligne par champ) ou Liste (puces <li>). Utilisez les flèches ▲ ▼ pour réordonner les champs. Le type Consigne importante génère un bandeau rouge. Le bouton + Image insère le code <img> de l'alarme orange. Cochez Réductible sur une section ajoutée pour un bloc <details>/<summary>. Couleurs et styles : panneau Mise en forme.

Texte de la remarque
