Créez vos sections, remplissez les champs, réglez la mise en forme et copiez le HTML généré.

Sections et champs

Chaque section a un titre et un type d'affichage : Inline (une ligne par champ) ou Liste (puces <li>). Utilisez les flèches ▲ ▼ pour réordonner les champs (l'ordre à l'écran = l'ordre dans le HTML généré). Le type Consigne importante génère un bandeau rouge avec icônes d'alarme. Le bouton + Image insère le code <img> de l'alarme orange.

Mise en forme

Espacement entre les lignes
0 = lignes collées. Appliqué en style inline (prioritaire sur les classes CSS).

Réglez pour chaque partie : couleur du texte, surlignement (fond), taille, gras, italique et souligné. Les titres de section apparaissent automatiquement ici.
